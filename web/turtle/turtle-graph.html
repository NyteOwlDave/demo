Purpose

These are some equations I copied from a FB video. The video animated the function plots and provided sound effects.

While I won't bother with either of those two enhancements, I do plan to convert these equation to JS and use my Turtle API to plot them.

The Turtle Graphics API has been extracted from the Turtle Demo. It's located in the api folder. The filename is turtle-graphics.js. It's being imported into this app already. All that remains is to write the app-specific logic that uses the turtle to draw wave-form plots.

The Archimedes Lab group on FB is the source. I used my MathJax app to code the Latex. Links to both are included in the References section.
